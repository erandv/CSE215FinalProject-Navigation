Please run these commands first in command prompt or pycharm's terminal. Please make sure you have the pip package installed.
pip install pyqt5
pip install haversine
pip install opencv-python

This program will teach us how a navigation app works. This one is designed for south Aerica as it has less countries and is easier to build the data base.
	User will be asked for his location and the destination. The output will show firstly the shortest path from his location to his destination and then, from a drop-down menu they can choose between different ways to get from their location to the destination.

For Pycharm, use the existing environment.
Please run the software file, not the main file.
